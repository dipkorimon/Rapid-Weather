# Rapid Weather
Weather application using real time API. 
Get the latest weather data for any region in the world.

# Tech Stack
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>Bootstrap</li>
  <li>JavaScript</li>
  <li>NodeJs</li>
  <li>ExpressJs</li>
</ul>
